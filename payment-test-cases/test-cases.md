Тест-кейсы

---

1. Позитивный сценарий

TC-01 — Успешная оплата картой

Предусловия:
Пользователь авторизован
В корзине есть товар
Доступен валидный номер карты

Шаги:

1. Перейти к оформлению заказа
2. Ввести корректные данные карты
3. Нажать «Оплатить»
4. Пройти 3-D Secure (если требуется)

Ожидаемый результат:

* Платеж проходит успешно
* Статус заказа = "Оплачен"
* Отображается сообщение об успешной оплате
* Пользователь получает email с чеком

---

2. Негативные сценарии

TC-02 — Отклонение платежа банком

Предусловия:
Используется карта с недостатком средств

Шаги:

1. Ввести данные карты
2. Нажать «Оплатить»

Ожидаемый результат:

* Платеж отклонен
* Отображается сообщение об ошибке ("Платеж отклонен банком")
* Статус заказа не изменяется на "Оплачен"

---

TC-03 — Ошибка платежного шлюза

Предусловия:
Платежный шлюз недоступен / возвращает ошибку

Шаги:

1. Ввести корректные данные
2. Нажать «Оплатить»

Ожидаемый результат:

* Отображается сообщение об ошибке ("Ошибка сервиса оплаты")
* Статус заказа остается прежним
* Пользователь может повторить попытку

---

TC-04 — Прерывание 3-D Secure

Предусловия:
Включен 3-D Secure

Шаги:

1. Ввести данные карты
2. Нажать «Оплатить»
3. Отменить операцию на странице 3-D Secure

Ожидаемый результат:

* Платеж не завершен
* Пользователь возвращается в приложение
* Статус заказа не меняется
