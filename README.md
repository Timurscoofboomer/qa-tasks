### 💳 1. Тест-кейсы для оплаты банковской картой
📁 Папка: `payment-test-cases`

**Условие:**
Веб-приложение позволяет пользователю оплатить заказ банковской картой через внешний платежный шлюз, включая:
- клиентскую валидацию
- отправку данных в платежный шлюз
- возможный редирект на 3-D Secure
- получение статуса платежа
- обновление статуса заказа
- отправку email-чека

**Реализовано:**
- ✅ Позитивные сценарии оплаты
- ❌ Негативные сценарии (отклонение, ошибки)
- ⚠️ Граничные значения
- 🔍 Проверка валидации формы
- 🔗 Проверка интеграции с платежным шлюзом

**Формат тест-кейсов:**
- ID
- Название
- Предусловия
- Шаги
- Ожидаемый результат

### 🪙 3. Задача по теории вероятности
📁 Папка: `coin-three-tosses`

**Условие:**
Монету подбрасывают три раза. Найти вероятность того, что выпадет ровно два орла.

**Решение:**
Рассматриваются все возможные исходы:
- Всего исходов: 8
- Подходящие: 3 (HHT, HTH, THH)

**Ответ:**
Вероятность = **3/8 = 0.375 (37.5%)**
